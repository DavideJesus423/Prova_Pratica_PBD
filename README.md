1. Modelagem do Sistema
  Modelo Conceitual: O desenho geral das entidades (Aluno, Professor, Disciplina e Turma) e como elas se ligam.
  Modelo Relacional:A estrutura lógica com as tabelas, chaves primárias (PK) e chaves estrangeiras (FK).

2.Scripts SQL
 Os ficheiros SQL incluídos fazem o seguinte:
   Criação: Cria o banco de dados e todas as tabelas com as regras de integridade.
 Inserção: Coloquei alguns dados de teste para que o sistema não fique vazio.
 Consulta: Um comando que gera a lista de alunos, as suas disciplinas e notas.
